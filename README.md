## ⚡ Key Features
- **Real-Time Dashboards**: Socket.IO-powered attendance and project monitoring.
- **Advanced Analytics**: Productivity scores based on project-wise employee performance.
- **Role-Based Access (RBAC)**: Distinct permissions for Admins, HR variants, Managers, and Employees.
- **Automated Attendance**: Multi-status tracking (Late, Half Day, Absent) with project mapping.
- **Project Management**: Isolated module for managing company-level deliverables.

## 🛠️ Local Development

### 1. Requirements
- Node.js (v18+)
- Firebase Account

### 2. Startup Commands
Run the backend:
```bash
cd Backend && npm start
```

Run the frontend:
```bash
npm run dev
```
